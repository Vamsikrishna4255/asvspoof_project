# 🔊 DeepFakeShield: Audio Deepfake Detection System

## 📌 Overview
DeepFakeShield is an AI-based system designed to detect whether an audio file is **Real or Fake** using deep learning techniques.  
The system aims to improve security in voice-based applications by identifying AI-generated or spoofed audio.

---

## 🚀 Features
- 🎯 Detects **Real vs Fake Audio**
- 🧠 CNN-based deep learning model (StableCNN)
- 📊 High performance (~0.98+ AUC)
- ⚖️ Handles class imbalance effectively
- 🎚️ Optimized threshold (0.24) for better prediction
- 🌐 User-friendly web interface
- 📩 Email alert for fake audio detection

---

## 🧠 Model Details
- Model Type: Convolutional Neural Network (CNN)
- Input: Audio converted to spectrogram
- Dataset Size: ~50,000 samples
- Output: Binary Classification (Real / Fake)
